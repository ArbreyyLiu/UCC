# Unified Compositional Controller (UCC) - Official Benchmark
This repository contains the COCO491-Enhanced and COCO491-Concise datasets used in the paper "Unified Compositional Controller: A Training-Free Framework for Highly Controllable Text-to-Image Generation".
## Dataset Structure
COCO491-Concise/: Contains the annotation files for the concise prompt set.
COCO491-Enhanced/: Contains the annotation files for the enhanced prompt set with regional descriptions.
## Usage
These files follow the standard COCO format and can be used to evaluate compositional generation capabilities.
## Code Release
The full source code for the UCC framework will be released here upon the acceptance of the paper.
